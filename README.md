# Hope Specialist Hospitals - Community Care Portal

Welcome to the official repository for the Hope Specialist Hospitals (HSH) Community Care Portal. This application serves as a comprehensive digital interface for patients and the community to access healthcare services, information, and resources.

## Project Overview

The HSH Community Care Portal is designed to provide:

- **Patient Information:** Detailed information about our doctors, specialties, and facilities.
- **Appointment Booking:** Easy access to schedule appointments and contact specialized departments.
- **News & Updates:** The latest news, events, and health tips from our medical experts.
- **Insurance Information:** A list of accepted insurance partners and coverage details.
- **Interactive Multimedia:** Features AI-themed visualizations and high-quality image galleries.
- **Legal & Accessibility:** Clear Terms of Service, Privacy Policy, and Accessibility statements.

## Technologies Used

This project is built using modern web technologies to ensure performance, accessibility, and scalability:

- **React:** For building a dynamic and responsive user interface.
- **JavaScript (ES6+):** Core development language.
- **Vite:** For fast development and optimized production builds.
- **Tailwind CSS:** For rapid, utility-first styling.
- **Shadcn UI:** for accessible and customizable UI components.
- **React Router:** For seamless client-side navigation.

## Key Features

- **AI Medical Scanner Simulation:** A custom CSS-based animation on the homepage demonstrating futuristic diagnostic capabilities.
- **Comprehensive Specialty Pages:** Detailed pages for various medical departments with relevant imagery.
- **Responsive Design:** Optimized for all device sizes from mobile phones to large desktop screens.

## Getting Started

To run this project locally:

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    ```
